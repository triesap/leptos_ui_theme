# leptos_ui_theme

An application-owned theming foundation for Leptos applications.

## Goals

- Provide a consistent way for applications to select and apply named themes.
- Keep theme definitions and application state app-owned.
- Complement UI kits without becoming a component registry or installer.
- Make theme behavior predictable for application authors and accessible UI.
